# PayFlow — платёжные страницы

Верстка интерфейса пополнения баланса: выбор способа оплаты (криптовалюта, карта, Perfect Money, MoneyGo, Skrill), выбор сети/монеты и экран с адресом депозита.

Демо для портфолио — без реальных платежей и без привязки к боевому сервису.

## Технологии

- HTML5
- CSS3 (адаптив)
- Vanilla JavaScript

## Как открыть

Откройте `index.html` в браузере или поднимите любой статический сервер в корне проекта:

```bash
npx serve .
```

## Структура

```
├── index.html                 # депозит криптовалютой
├── cards-template/
│   ├── master-card.html
│   ├── perfect-money.html
│   ├── money-go.html
│   └── skrill.html
├── js/script.js               # выбор метода и шаги депозита
├── .nojekyll                  # нужен для GitHub Pages
└── static/assets/
    ├── css/                   # main.css + layout.css
    └── img/
```

## Возможности интерфейса

- переключение способов оплаты между страницами;
- выбор криптовалюты и сети (например USDT BEP-20 / ERC-20 / TRC-20);
- пошаговый сценарий: список → сеть → адрес и QR;
- шапка, футер и адаптивная вёрстка.
